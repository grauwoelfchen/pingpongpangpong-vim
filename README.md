# pinponpanpon-vim

Let's check japanese weather warning Announce in Vim.

pinponpanpon ♪

## Require

* [webapi-vim](https://github.com/mattn/webapi-vim)

## Usage

```
:PinponPanpon さいたま市

or set in .vimrc

g:pinponpanpon_area_name = 'さいたま市'
```
## Source

* [Japan Meteorological Agency](http://www.jma.go.jp/jp/warn/)
